# Credit Risk Prediction with Explainable AI

An end-to-end machine learning project that predicts borrower default risk while providing clear, interpretable explanations for every prediction.

🚀 **Experience the Model in Action:** [Open Live App](https://s58ugqceqqkanyg3a8yfe5.streamlit.app)

---

## What This Does
- Predicts **probability of loan default** using financial & behavioral data  
- Explains predictions using **SHAP (feature-level insights)**  
- Provides an **interactive interface** for real-time risk assessment  

---

## Why It Matters
Traditional credit models often lack transparency. This project combines **strong predictive performance** with **explainability**, making it suitable for real-world financial decision-making where trust and accountability are critical.
